# Security

This repository contains public architecture guidance, not deployable product code.

Report security issues that affect the repository content by opening a GitHub issue. Do not include secrets, customer data, private tenant details, or non-public Microsoft information in an issue.

If you believe the issue describes a vulnerability in a Microsoft product or service, report it through the Microsoft Security Response Center instead:

[Report a vulnerability to MSRC](https://msrc.microsoft.com/report)
