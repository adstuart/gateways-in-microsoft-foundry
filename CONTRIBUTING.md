# Contributing

Contributions are welcome when they improve public, source-backed guidance.

## Content rules

- Use public sources only.
- Link to current Microsoft Learn or vendor documentation for product claims.
- Do not include customer names, private tenant details, internal URLs, credentials, or non-public roadmap information.
- Keep guidance concise and architecture-focused.
- Use sentence-style headings.
- Use Microsoft terminology, including Microsoft Entra ID, sign in, allowlist, blocklist, and use.
- Avoid hype, filler, and unsupported claims.

## Pull request checklist

- [ ] The change uses public sources.
- [ ] The change does not include secrets or private details.
- [ ] Product availability, tier support, and limits were checked against current documentation.
- [ ] New guidance is linked from the README when appropriate.
- [ ] Mermaid diagrams still render on GitHub.
